Class FB.Async.Data

Known Subclasses:
FB.Data.Query
An node that holds data that may not be available immediately

Constructor

FB.Async.Data ( value )
Parameters:
value <object> [Optional] value of the data, if available.

Methods

set

void set ( Set )
Parameters:
Set <object> value property of the data object. This will cause "value" event to be fire on the object. Any callback functions that are waiting for the data through wait() methods will be invoked if the value was previously not set.
Returns: void

wait

void wait ( A )
Wait until this.value is set. Example:
var friendInfos = FB.Data.query( 'select name, pic from user where uid in (select uid2 from {0})', friends); friendInfos.wait(function(data) { // Render info. For illustration of API, I am using any XFBML tags var html = ''; FB.forEach(data, function(info) { html += '<p>' + info.name + '<img src="' + info.pic + '" /></p>'; }); FB.$('infos').innerHTML = html; });
Parameters:
A <function> callback function that will be invoked when this.value is set.
Returns: void
